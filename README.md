scala.js-starter [![build](https://travis-ci.org/daggerok/scala.js-starter.svg?branch=master)](https://travis-ci.org/daggerok/scala.js-starter)
================

Scala.js web application starter

- using gradle

```sh
gradle clean test run
```

- using sbt

```sh
sbt clean test run
```
